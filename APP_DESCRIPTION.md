# Moonlight V+ for HarmonyOS — 应用商店介绍

## 一句话简介

低延迟串流，把你的 PC 游戏随时带到鸿蒙设备上畅玩。

## 详细介绍

### Moonlight V+ for HarmonyOS — 专为鸿蒙打造的 PC 游戏串流客户端

基于 Moonlight 开源协议，Moonlight V+ for HarmonyOS 是 HarmonyOS NEXT 上首款原生高性能游戏串流应用。从你的 PC 到手机、平板，毫秒级延迟，零妥协画质。

**⚡ 原生高性能，不是移植**
专为 HarmonyOS 从零构建，深度适配系统能力。硬件加速解码，帧级调度优化，串流延迟低至个位数毫秒。

**🎮 手柄体验，超越本地**
- GC Kit 与蓝牙手柄即连即用，按键映射开箱完整
- **摇杆模拟鼠标** — 不接鼠标也能操控桌面和策略游戏
- **手柄陀螺仪瞄准** — 传感器数据实时透传，体感射击精度媲美原生
- 手柄振动反馈同步，力度可调
- Home 键防劫持，蓝牙手柄误触不再退出应用

**🖥️ 画面自适应，横竖屏无缝切换**
- 最高 4K@120fps + HDR，发挥硬件全部潜力
- **服务端旋转实时同步** — 主机切换横竖屏，客户端自动跟随，零拉伸
- 等比/拉伸双模式，任意屏幕比例完美适配

**🌈 HDR 独家体验**
- HDR10 + HLG (HDR Vivid) 双协议支持
- 华为设备原生 HDR Vivid 适配，色彩准确度远超普通 HDR10
- 搭配 Foundation Sunshine 使用效果更佳

**🖥️ 虚拟显示器支持**
- 串流专用虚拟屏幕，不占用你的主显示器
- 独立分辨率和刷新率设置，串流不影响本地使用

**👆 触控深度定制**
- 屏幕虚拟手柄：透明度、位置、布局随心调
- 增强触控：支持分区映射、速度曲线、长按识别
- 双击拖动等高级手势，触屏也能精准操作

**🔊 音频全覆盖**
- 立体声 / 5.1 / 7.1 环绕声
- HarmonyOS 5.0+ 空间音频支持
- 麦克风透传，游戏内语音开箱即用

**📊 延迟可视，一切透明**
- 实时性能覆盖层：解码/网络/主机延迟、帧率、码率全量展示
- 覆盖层可拖拽、可锁定、显示项目可自定义
- 串流结束自动汇总平均延迟，帮你定位瓶颈

**🎙️ 更多你会用到的**
- WAN 远程串流，出门在外也能连回家里 PC
- 自动发现局域网主机，配对即玩
- 串流中动态调节码率，无需断开重连
- 服务端指令执行，远程操控更灵活
- 断线智能退出，不留后台空转
- AES-128 加密连接，完全本地通信
- 无广告，不采集任何数据

**系统要求**
- HarmonyOS NEXT 5.0 或更高版本
- 支持设备：华为手机 / 平板 / MatePad

**服务端兼容**
- Foundation Sunshine（推荐，支持全部增强特性）
- LizardByte Sunshine
- NVIDIA GeForce Experience（仅基础功能）

仅需鸿蒙设备 + 同网段或公网可达的 PC 即可开始。
