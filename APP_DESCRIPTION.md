## 详细介绍

### Moonlight V+ for HarmonyOS

基于 Moonlight 开源协议的鸿蒙原生 PC 游戏串流客户端，打通视觉、听觉、触觉全链路，带来真正的沉浸式游戏体验。

**原生性能**
从零面向 HarmonyOS 构建，硬件加速解码 + 帧级调度优化，串流延迟低至个位数毫秒，120fps 满帧流畅不掉帧。

**HDR Vivid 原生适配**
鸿蒙设备用 HDR Vivid (HLG) 才是对的。普通 HDR10 在手机上色彩映射先天受限，高光容易过曝，暗部容易糊。HDR Vivid 走的是鸿蒙原生显示管线，色准、亮度映射、暗部细节都有质的区别。我们同时支持 HDR10 和 HDR Vivid 双协议——但用过 Vivid 之后大概不会想切回去。

**画面**
- 最高支持 4K@120fps + HDR，帧跑满
- 横竖屏旋转自动跟随服务端同步，不会拉伸变形
- 等比与拉伸双模式适配

**手柄支持**
- 最多同时连接 4 个手柄，支持本地多人游戏
- 内置 700+ 手柄映射数据库（Xbox / PS / Switch Pro / 8BitDo 等主流型号开箱即用）
- GC Kit 与蓝牙手柄即连即用，按键映射无需手动配置
- 摇杆模拟鼠标，不接鼠标也能操作桌面和策略游戏
- 陀螺仪瞄准：传感器数据实时透传，体感射击确实能用
- Home 键防误触，蓝牙手柄不会莫名其妙把你弹出去了

**音频振动**
分析串流音频的低频能量与节拍信息，直接驱动手机马达产生振动反馈——不需要手柄。
- 三种模式：游戏/影视追踪低频冲击，音乐/节奏追踪鼓点节拍，自动模式根据音频特征切换
- 振动响应延迟低于一帧，强度三档可调

**画面**
- 最高支持 4K@120fps + HDR，帧跑满，低延迟
- 横竖屏旋转自动跟随服务端同步，不会拉伸变形
- 等比与拉伸双模式适配

**音频**
- 立体声 / 5.1 / 7.1 环绕声全覆盖
- 空间音频支持（HarmonyOS 5.0+）
- 麦克风透传，语音功能开箱即用

**其他**
- 实时性能覆盖层：延迟、帧率、码率一目了然
- WAN 远程串流 / 局域网自动发现
- 虚拟显示器，串流不占用主屏幕
- 屏幕虚拟手柄，布局位置可自定义
- 串流中动态调节码率，无需重连
- 全链路加密，无广告，不采集任何数据

**系统要求**
- HarmonyOS NEXT 5.0+
- 华为手机 / 平板 / MatePad

**服务端兼容**
- Foundation Sunshine（推荐，增强特性全支持）
- LizardByte Sunshine
- NVIDIA GeForce Experience（基础功能）

一台鸿蒙设备，加上同网段或公网可达的 PC，就够了。
